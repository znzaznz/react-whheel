webpack-dev-server用来做什么  
开一个server,默认监听8080端口 将文件放在内存内，这样做会很快

package.json script写命令：
```json
{
  "scripts" : {
    "start": "webpack-dev-server",
    "build": "webpack"
  }
}
```
